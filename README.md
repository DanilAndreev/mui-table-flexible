# mui-table-flexible
## Overview
__mui-table-flexible__ - table processor, designed to create tables with resize handlers for big amount of data.


## Getting started
### Installation
Using __npm__: ```npm install mui-table-flexible```  
Using __yarn__: ```yarn add mui-table-flexible```
### Usage
```jsx
    add usage
```
